# Cheshman - فروشگاه تخصصی عینک

پروژه مدیریت و فروشگاه تخصصی عینک با طراحی Neo-Brutalism

## نصب و راه‌اندازی

```bash
npm install
npm run dev
```

## ساختار پروژه

- `src/` - کد منبع
  - `js/` - فایل‌های JavaScript
  - `css/` - فایل‌های CSS
  - `pages/` - صفحات مختلف
  - `assets/` - تصاویر و منابع
- `index.html` - صفحه اصلی

## ویژگی‌های پروژه

✅ Landing Page با Interactive Eye Tracker
✅ Store - فروشگاه محصولات
✅ Cart - سبد خریدی
✅ Admin Panel - پنل مدیریت
✅ Neo-Brutalism Design
✅ RTL Layout
✅ Tailwind CSS
✅ Vite Build Tool

## فناوری‌های استفاده شده

- Vite
- Tailwind CSS
- Vanilla JavaScript
- HTML5
